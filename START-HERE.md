## ⚡ Super Quick Start (If You're Impatient)

```bash
# 1. Install
cd /Users/ashborn/.openclaw/workspace/sce-construction-manager
npm install

# 2. Configure
cp .env.example .env
nano .env
# Add your OpenAI API key and Google credentials

# 3. Initialize Google Sheets
npm run setup:sheets
# Copy the output IDs back into .env

# 4. Start
npm run dev
# Visit http://localhost:3000

# 5. Create your first project!
```

**Done! Now read USER-GUIDE.md to learn everything else.**

## 🎯 What This System Does

1. **Project Intake** - Capture all info in one form
2. **AI Photo Analysis** - Automatically generate scope of work
3. **Cost Estimation** - AI calculates detailed bids
4. **Bid Generation** - Professional PDFs with SCE branding
5. **Vendor Recommendations** - Smart matching by trade/location/reliability
6. **Dashboard** - Track all 8+ jobs from one screen
7. **WhatsApp Monitoring** - Auto-update vendor data from messages
